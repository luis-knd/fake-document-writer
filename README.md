## Fake Document Writter :page_with_curl: :writing_hand:
Fake document writter is a program to write a random text file for a period indicated.


### Technical Requirements.
You need have installed the following program in your computer to run the application successfully:
- Python

Validate the correct python installation in your system
```shell
python --version
```
If the command not returned the version, [install python in your system](https://www.python.org/downloads/)



### How to run the application step by step.
**1.** In the main path run the next command:
```shell
python main.py 60
```
In this example the last parameter (`60`) represent the time in second while the program write fake text. 

The output text will be generated in the file: `file_generator.txt`.
